VEG DDF classes

This page links to the DDF files that are in development for the VEG specification. These files can be found in the DDF project of the OSGi build tree.

These files are all XML, however, an XSLT stylesheet is supplied that will format the files for human consumption. This processing happens in the browser. It is tested in IE 6 and Firefox.

File Description
vehiclebody.xml Vehicle Body Managed Objects
vehiclechassis.xml Vehicle Chassis Managed Objects
vehicleinterior.xml Vehicle Interior Managed Objects
vehiclepowertrain.xml Vehicle Power Train Managed Objects
