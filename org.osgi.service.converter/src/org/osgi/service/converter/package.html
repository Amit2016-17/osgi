Formatter Package Version 1.0.

The purpose of this package is to provide access to a Converter service. Bundles wishing to use this package must list the package in the Import-Package header of the bundle's manifest. For example: 

Consumers: Import-Package: org.osgi.service.converter; version="[1.0,2.0)"
Providers: Import-Package: org.osgi.service.converter; version="[1.0,1.2)"
