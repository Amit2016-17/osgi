The OSGi DMT Specification Version 1.0.

This package contains the public API of the Device Management service. Permission classes are provided by the org.osgi.service.dmt.security package, and DMT plugin interfaces can be found in the org.osgi.service.dmt.spi package.

Bundles wishing to use this package must list the package in the Import-Package header of the bundle's manifest. For example: 

Import-Package: org.osgi.service.dmt; version=1.0
